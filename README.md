# AppManager
> Step 1. Add the JitPack repository to your build file
 ``` Project Gradle
  allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
 ```
> Step 2. Add the dependency
``` Module Gradle
dependencies {
	        implementation 'com.github.PakMasterDeveloper:AppManager:1.0'
	}
```
